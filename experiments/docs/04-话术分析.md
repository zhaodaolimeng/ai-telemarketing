# 话术有效性分析

## 分析目标
1. **识别有效话术** - 哪些话术与高回款率相关
2. **识别无效/负面话术** - 哪些话术应该避免
3. **发现话术模式** - 有效话术有什么共同特征
4. **生成话术建议** - 为不同场景推荐话术

---

## 话术有效性度量

### 关联指标

| 指标 | 定义 | 说明 |
|------|------|------|
| **话术成功率** | 使用该话术后续获得承诺的比例 | 短期效果 |
| **话术回款率** | 使用该话术最终回款的比例 | 长期效果 |
| **客户接受度** | 使用后客户态度变好的比例 | 即时反应 |
| **提升率** | 相比不用该话术的效果提升 | 因果推断 |

### 统计显著性检验
- p-value < 0.05 视为显著
- 使用Bootstrap计算置信区间
- 控制混淆变量（客户风险、金额等）

---

## 分析维度

### 1. 按话术功能分类
| 功能类型 | 例子 | 预期效果 |
|---------|------|---------|
| **共情安抚** | "Saya mengerti situasi Bapak..." | 中等 |
| **轻度提醒** | "Ini hanya pengingat..." | 低 |
| **中度施压** | "Ini akan mempengaruhi kredit..." | 中等 |
| **引导承诺** | "Kapan bisa bayar?" | 高 |
| **确认承诺** | "Jadi tanggal 15 ya?" | 高 |
| **强硬警告** | "Kami harus tindak lanjut..." | 混合 |

### 2. 按表达风格分类
| 风格 | 例子 |
|------|------|
| **直接型** | "Anda harus bayar sebelum 30 April" |
| **间接型** | "Kalau bisa, sebaiknya dibayar sebelum 30 April" |
| **问句引导** | "Kapan Bapak rasa bisa bayar?" |
| **陈述句** | "Tanggal 30 adalah batasnya" |
| **多问少答** | 频繁提问获取信息 |
| **多说少问** | 更多陈述信息 |

### 3. 按施压强度分类
- L1-L5 强度，看不同强度的效果

## 有效话术特征提取

### 步骤1: 提取候选话术
- 从成功案例中提取高频句子
- 聚类相似句子得到候选模板
- 人工筛选有业务意义的

### 步骤2: 评估效果
```python
for utterance_template in candidate_templates:
    # 找到使用了相似话术的对话
    matches = find_similar_utterances(utterance_template)

    # 计算指标
    success_rate = calc_success_rate(matches)
    repaid_rate = calc_repaid_rate(matches)

    # 与基线比较
    baseline = calc_baseline(similar_cases)
    lift = success_rate - baseline

    # 统计检验
    p_value = significance_test(matches, baseline)

    # 记录
    results.append({
        'template': utterance_template,
        'success_rate': success_rate,
        'repaid_rate': repaid_rate,
        'lift': lift,
        'p_value': p_value,
        'count': len(matches)
    })
```

### 步骤3: 生成话术库
- 按效果排序
- 过滤显著正提升的
- 人工审核后整理成话术库

## 话术模板格式

```yaml
- id: UTTR_001
  text: "Kapan Bapak/Ibu kira-kira bisa membayarnya?"
  state: commitment_extract
  language: id
  intensity: L2
  effect:
    success_rate: 0.68
    repaid_rate: 0.52
    lift: +0.15
    p_value: 0.002
  tags:
    - 问句
    - 引导承诺
    - 尊重
  variations:
    - "Tanggal berapa Bapak/Ibu bisa bayar?"
    - "Kapan kira-kira bisa selesai pembayarannya?"
  when_to_use:
    - 客户已有还款意愿
    - 对话中后期
    - 客户没有明确拒绝
  when_not_to_use:
    - 客户刚表示经济困难
    - 客户情绪激动
```

## 对比分析：成功 vs 失败对话

### 成功对话常见特征
1. **更多引导性提问**
   - "Kapan bisa?" vs "Anda harus bayar sekarang"

2. **适当的共情后再引导**
   - "Saya mengerti... tapi..."

3. **获取具体的承诺**
   - "Jadi tanggal 15, 500 ribu ya?"

4. **说话比例平衡**
   - 不是催收员一直说

### 失败对话常见特征
1. **一开始就高强度施压**
2. **不听客户解释就打断**
3. **没有获得具体承诺就结束**
4. **话术重复啰嗦**
