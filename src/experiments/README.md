# 催收对话数据分析实验

## 实验目的

基于真实场景的催收语音对话数据以及后续回款状态，进行数据分析以发现：
1. **真实会话状态** - 催收对话中实际存在的核心状态
2. **有效话术识别** - 哪些话术能有效促进用户还款
3. **对话模式分析** - 成功催回的对话有什么特征
4. **打断时机规律** - 什么时候打断效果最好

## 数据要求

### 原始数据
| 数据项 | 说明 | 格式 |
|--------|------|------|
| 语音文件 | 催收通话录音 | WAV/MP3 |
| 回款状态 | 是否还款、时间、金额 | CSV/JSON |
| 元数据 | 案件信息（逾期天数、金额等） | CSV/JSON |

### 标注数据（可选但推荐）
- 人工标注的会话状态
- 人工标注的关键转折点
- 人工标注的有效话术

---

## 实验流程

```
1. 数据准备
   ↓
2. 语音转写 + 说话人分离
   ↓
3. 对话结构化（时间线、角色分离）
   ↓
4. 标注或半监督学习发现会话状态
   ↓
5. 话术效果分析（关联回款状态）
   ↓
6. 发现有效模式、生成报告
```

## 核心分析目标

### 1. 会话状态发现
- 对话中有哪些典型状态？
- 状态如何转移？
- 哪些状态是关键转折点？

### 2. 话术有效性分析
- 哪些话术与高回款率正相关？
- 哪些话术与低回款率相关？
- 不同客户/场景下的有效话术差异？

### 3. 对话节奏分析
- 打断时机的影响？
- 语速、停顿的影响？
- 谁主导对话更好？
