# 业务背景

## 项目概述
本项目是一个AI智能催收对话分析系统，通过分析印尼语催收电话录音，识别成功催收的关键话术和对话模式。

## 催收环节说明

### seats_name 字段含义
- **H2**: 最早的催收环节
  - 客户刚逾期不久
  - 还款意愿相对较高
  - 成功率较高
- **H1**: 中期催收环节
  - 逾期一段时间
  - 还款意愿中等
- **S0**: 最迟/困难催收环节
  - 逾期较长时间
  - 客户还款意愿较低
  - 常遇到强硬拒绝还款的客户
  - 催收难度最大

### 分析注意事项
在进行成功/失败对话对比时，**必须尽可能在相同催收环节内横向比较**，避免因不同环节客户还款意愿天然差异导致分析偏差。

## 标签说明

### repay_type 字段
- `repay`: 24小时内成功还款
- `extend`: 协商延期还款
- `NaN/未标注`: 24小时内未还款（沟通失败）

### result 分类
- `success`: 成功（repay 或 extend）
- `failed`: 失败（未标注/24小时未还款）

## 数据过滤规则

### 需要过滤的对话类型
1. **语音信箱自动回复**
   - 特征：彩铃后直接转语音信箱
   - 典型话术："silakan tinggalkan pesan setelah bunyi bip"（请在哔声后留言）
   - 处理方式：直接从分析样本中剔除

2. **无效对话**
   - 特征：无人接听、电话中断、客户直接挂断
   - 处理方式：标记为无效，不参与话术分析

## 分析目标
1. 识别成功催收的关键话术和关键词
2. 总结有效的对话流程模式
3. 为AI智能催收系统提供话术建议
