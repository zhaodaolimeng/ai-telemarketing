## 二、完整对话流程模板

### 模板1: 标准成功流程（推荐）

```indonesian
[1] AGENT: Halo, selamat sore Pak/Bu.
    CUSTOMER: [回应]

    CUSTOMER: Ya/Benar
[3] AGENT: Untuk pinjaman yang harus dibayar, kapan bisa bayar Pak/Bu?
    CUSTOMER: [时间回应]
[4] AGENT: Oke, jam [X] ya Pak/Bu. Saya tunggu.
    CUSTOMER: Oke/Ya
[5] AGENT: Ya, ya. Baik, terima kasih Pak/Bu.
    CUSTOMER: Terima kasih
[6] AGENT: Selamat sore.
```

## 四、对话状态转移设计

### 状态机定义

```
START
  ↓
[状态1: GREETING] 开场问候
  ├─ 成功回应 → 进入 IDENTIFY
  └─ 无人接听 → 结束
  ↓
[状态2: IDENTIFY] 身份确认
  ├─ 确认成功 → 进入 PURPOSE
  └─ 找不到人 → 尝试稍后联系
  ↓
[状态3: PURPOSE] 说明目的
  ├─ 客户理解 → 进入 ASK_TIME
  └─ 客户不理解 → 重复说明
  ↓
[状态4: ASK_TIME] 询问还款时间
  ├─ 客户给出时间 → 进入 COMMIT_TIME
  ├─ 客户含糊 → 追问具体时间
  └─ 客户拒绝 → 进入 NEGOTIATE
  ↓
[状态5: COMMIT_TIME] 约定具体时间
  ├─ 客户同意 → 进入 CONFIRM
  └─ 客户犹豫 → 给出建议时间
  ↓
[状态6: CONFIRM] 获得确认
  ├─ 明确确认 → 进入 CLOSING
  └─ 确认模糊 → 再次确认
  ↓
[状态7: CLOSING] 礼貌结束
  ↓
END
```

---

## 五、各状态的标准话术

### 状态1: GREETING 开场问候

**触发**：电话接通

**标准话术**：
```indonesian
"Halo, selamat [pagi/siang/sore] Pak/Bu."
"Hello, selamat [pagi/siang/sore]."
```

**回应检测**：
- "halo", "selamat", "ya", "pagi", "sore"

---

### 状态2: IDENTIFY 身份确认

**触发**：客户有回应问候

**标准话术**：
```indonesian
"Saya dari aplikasi Extra."
"Dengan Bapak/Ibu [Nama] ya?"
"Ya Pak/Bu [Nama]?"
```

**回应检测**：
- "ya", "benar", "iya", "betul"

---

### 状态3: PURPOSE 说明目的

**触发**：身份确认成功

**标准话术**：
```indonesian
"Untuk pinjaman yang harus dibayar."
"Untuk pembayaran pinjaman, Pak/Bu."
```

**避免**：
```indonesian
❌ 不要过度解释应用
```

---

### 状态4: ASK_TIME 询问还款时间

**触发**：客户理解目的

**标准话术**：

```indonesian

"Jadi jam berapa ya?"
```

---

### 状态5: COMMIT_TIME 约定具体时间

**触发**：客户给出时间

**标准话术**：
```indonesian
"Oke, jam [X] ya Pak/Bu."
"Jam [X] ya, saya tunggu."
"Maksimal jam [X] hari ini ya."
"Siap, saya tunggu jam [X]."
```

**关键**：必须包含 "saya tunggu"

---

### 状态6: CONFIRM 获得确认

**触发**：约定时间后

**标准话术**：
```indonesian
"Ya, ya."
"Oke."
"Baik."
"Siap."
"Jadi jam [X] ya Pak/Bu, saya tunggu."
```

**回应检测**：
- "oke", "ya", "siap", "baik"

---

### 状态7: CLOSING 礼貌结束

**触发**：获得明确确认后

**标准话术**：
```indonesian

"Terima kasih."
"Selamat [pagi/siang/sore]."
"Siap, sampai jumpa."
```

## 六、成功话术总结

### TOP 5 最有效话术（必须包含）

1. **"Saya tunggu."** - 表达期待，建立责任感
2. **"Jam [X] ya Pak/Bu."** - 明确约定具体时间
3. **"Ya, ya."** - 多次确认，增强承诺感
4. **"Dengan Bapak/Ibu [Nama] ya?"** - 礼貌称呼，建立尊重
5. **"Terima kasih Pak/Bu."** - 礼貌结束，保持关系

---

### 话术选择原则

**优先级从高到低**：
1. 包含 "saya tunggu" 的话术
2. 包含具体时间 "jam [X]" 的话术
3. 使用确认用语 "oke/ya/iya" 的话术
4. 使用礼貌称呼 "Pak/Bu" 的话术
5. 表达感谢的话术

## 七、常见问题处理

### Q1: 客户说 "aman"（没问题）但没说时间

**不要**：
```indonesian
❌ AGENT: Oke aman.
```

**应该**：
```indonesian
✅ AGENT: Oke, jam berapa tepatnya Pak/Bu?
✅ AGENT: Jadi jam berapa ya, saya tunggu?
```

---

### Q2: 客户只说 "hari ini"（今天）

**不要**：
```indonesian
❌ AGENT: Oke hari ini.
```

**应该**：
```indonesian
✅ AGENT: Oke, hari ini jam berapa ya Pak/Bu?
✅ AGENT: Jam berapa hari ini, saya tunggu?
```

---

### Q3: 客户问很多关于应用的问题

**不要**：
```indonesian
❌ AGENT: [详细解释应用]
```

**应该**：
```indonesian
✅ AGENT: Ya Pak/Bu, kita fokus ke pembayaran dulu ya. Kapan bisa bayar?
```
