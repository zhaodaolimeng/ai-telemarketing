# 智能语音催收话术开发指南

## 项目目标回顾

本项目的核心目标是：**识别能提高催回率的有效催收话术**，用于智能语音催收系统的开发。

**关键原则**：
- 重点关注**催收员**的话术，客户的回应主要用于理解对话状态
- 识别成功催收的对话模式和关键话术
- 输出可直接用于智能语音开发的话术模板

---

## 关键发现

### 1. 成功对话的核心特征

| 特征 | 说明 |
|------|------|
| **confirmation** | 多次使用确认用语（oke/ya/iya） |
| **identify_caller** | 主动确认对方身份（Bapak/Ibu/Pak/Bu） |
| **greeting** | 礼貌的问候（selamat pagi/siang/sore） |
| **commit_time** | 明确约定还款时间（commit_time 成功21次 vs 失败3次） |

### 2. 最有效的催收员话术

**高成功率短语**（成功样本中显著更多）：
- `"saya tunggu"` - 我等待（成功9次，失败0次）
- `"dengan bapak"` - 与先生（成功7次，失败0次）
- `"ya, ya."` - 是的（成功7次，失败0次）
- `"terima kasih."` - 谢谢.（成功8次，失败1次）

**失败导向短语**（失败样本中显著更多）：
- `"jam 3"` - 3点（成功0次，失败5次）
- `"ya pak"` - 是的先生（成功2次，失败4次）
- `"hari ini"` - 今天（成功4次，失败5次）

## 对话状态分析

### 理想对话流程

```
[状态1: 开场]
  AGENT: Halo, selamat sore Pak/Bu
  ↓
[状态2: 身份确认]
  AGENT: Dengan Bapak/Ibu [Nama] ya?
  AGENT: Saya dari aplikasi Extra
  ↓
[状态3: 说明目的]
  AGENT: Untuk pinjaman yang harus dibayar
  ↓
[状态4: 约定时间]
  AGENT: Kapan bisa bayar? Jam berapa?
  AGENT: Jam [X] ya Pak/Bu
  ↓
[状态5: 获得确认]
  AGENT: Oke, saya tunggu jam [X]
  ↓
[状态6: 结束]
  AGENT: Terima kasih Pak/Bu, selamat sore
```

### 关键状态转移

- **开场 → 确认身份**: 必须用礼貌称呼
- **确认身份 → 约定时间**: 简洁说明目的，不啰嗦
- **约定时间 → 获得确认**: 必须得到明确的 "oke/ya"
- **获得确认 → 结束**: 表达感谢，保持礼貌

## 智能语音话术模板

### 模板1: 标准开场

```indonesian
AGENT: Halo, selamat [pagi/siang/sore] Pak/Bu.
AGENT: Saya dari aplikasi Extra.
AGENT: Dengan Bapak/Ibu [Nama] ya?
```

### 模板2: 确认还款时间

```indonesian
AGENT: Untuk pinjaman yang harus dibayar, kapan bisa bayar Pak/Bu?
AGENT: Jam berapa ya?
[客户回应后]
AGENT: Oke, jam [X] ya Pak/Bu. Saya tunggu.
AGENT: Terima kasih konfirmasinya.
```

### 模板3: 引导确认

```indonesian
AGENT: Jadi jam [X] hari ini ya Pak/Bu?
AGENT: Saya tunggu pembayaran jam [X].
AGENT: Oke, baik. Terima kasih.
```

### 模板4: 结束对话

```indonesian
AGENT: Siap, saya tunggu jam [X].
AGENT: Terima kasih Pak/Bu.
AGENT: Selamat [pagi/siang/sore].
```

## 需要避免的话术

### ❌ 不要过度强调应用

```indonesian
❌ Salah: "Untuk aplikasi, wajib dari aplikasi"
❌ Salah: "Aplikasi Extra uang"
✅ Benar: "Saya dari aplikasi Extra untuk pinjaman"
```

### ❌ 不要只用被动语态

```indonesian
❌ Salah: "Dibayar jam berapa?"
✅ Benar: "Kapan bisa bayar, Pak/Bu?"
```

### ❌ 不要接受模糊回应

```indonesian
❌ Salah: "Oke aman" (如果客户只说 "aman")
✅ Benar: "Jam berapa tepatnya, Pak/Bu?"
```

### ❌ 不要过度简短

```indonesian
❌ Salah: "Ya pak" (只用这个回应)
✅ Benar: "Oke Pak, saya tunggu jam 5"
```

## 对话示例

### 成功对话示例

```indonesian
[1] AGENT: Halo Selamat Sore Pak
    CUSTOMER: Sore
[2] AGENT: Ya Pak Rilan tau kurba dari uang sama ekstra
    CUSTOMER: Oh ingatnya lagi belum lupa jam 5 sudah dibayar ya Pak
[3] AGENT: Siap siap
    CUSTOMER: Aksi kerja makanya Bu
[4] AGENT: Ya
    CUSTOMER: Bagaimana?
[5] AGENT: Aksi kerja belum bisa keluar
    CUSTOMER: Oke jam 5 maksimal ya Pak, terang kerja samannya ya
[6] AGENT: Pasti hari ini pasti saya kelimlah
    CUSTOMER: Ya jam 5 Pak, saya komitmen awal
[7] AGENT: Baik saya tunggu ya
    CUSTOMER: Oke terima kasih, sama Sore
```

**成功关键点**：
- 使用了 "saya tunggu"
- 明确约定了 "jam 5"
- 多次确认 (ya/siap/baik/oke)
- 礼貌结束 (terima kasih)

### 失败对话示例

```indonesian
[1] AGENT: Halo, selamat pagi.
    CUSTOMER: Ya.
[2] AGENT: Bapak Maruzuki.
    CUSTOMER: Ya.
[3] AGENT: Saya dari aplikasi ekstra, Pak. Ini tagian yang bisa dibayar.
    CUSTOMER: Tagian apa?
[4] AGENT: Di sini adalah tagian atas nama Bapak Maruzuki.
    CUSTOMER: Aplikasi ekstra uang no, Pak.
[5] AGENT: Terima kasih.
```

**失败原因**：
- 客户明确否认债务（"uang no"），坐席直接放弃，未做任何挽留
- 没有使用任何追问策略（如"记得有借款吗？"、"之前还款记录显示..."等）
- 没有尝试约定还款时间
- 没有使用 "saya tunggu" 或其他施压话术
- 对话仅5轮即草草结束，未达成任何目标

## 智能语音系统设计建议

### 1. 对话状态机设计

```python
states = [
    "GREETING",        # 开场问候
    "IDENTIFY",        # 确认身份
    "PURPOSE",         # 说明目的
    "ASK_TIME",        # 询问还款时间
    "COMMIT_TIME",     # 约定具体时间
    "CONFIRM",         # 获得确认
    "CLOSING"          # 结束对话
]
```

### 2. 关键触发词检测

在每个状态，检测客户的关键回应：

| 状态 | 需要检测的客户回应 |
|------|------------------|
| GREETING | "halo", "selamat", "ya" |
| IDENTIFY | "ya", "benar", "iya" |
| ASK_TIME | "jam [X]", "pukul [X]", "hari ini" |
| CONFIRM | "oke", "ya", "siap", "baik" |

### 3. 话术选择优先级

当有多个可选话术时，按以下优先级：
1. 使用包含 "saya tunggu" 的话术
2. 使用包含具体时间点 "jam [X]" 的话术
3. 使用确认用语 "oke/ya/iya"
4. 使用礼貌称呼 "Pak/Bu"
