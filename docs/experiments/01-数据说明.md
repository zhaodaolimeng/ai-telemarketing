# 数据说明

## 数据输入格式

### 1. 语音数据
```
data/raw/
├── case_001.wav
├── case_002.wav
├── case_003.wav
└── ...
```
- 格式：WAV (推荐) 或 MP3
- 采样率：16kHz 或更高
- 声道：单声道或立体声

### 2. 案件元数据
**文件**: `data/raw/cases.csv`

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| case_id | string | ✅ | 案件ID，与语音文件名对应 |
| amount | float | ✅ | 欠款金额 |
| days_overdue | int | ✅ | 逾期天数 |
| stage | string | ✅ | M0/M1/M2+ |
| repaid | bool | ✅ | 是否回款 |
| repaid_amount | float | ❌ | 回款金额 |
| repaid_days | int | ❌ | 对话后多少天回款 |
| customer_name | string | ❌ | 客户姓名（脱敏） |

### 3. 人工标注（可选）
**文件**: `data/annotations/annotations.json`

```json
{
  "case_001": {
    "turns": [
      {
        "start_time": 0.5,
        "end_time": 2.3,
        "speaker": "agent",
        "text": "Selamat siang, Bapak John...",
        "state_label": "opening",
        "effectiveness": 3
      }
    ],
    "key_points": [
      {"time": 45.2, "description": "客户开始松动"},
      {"time": 120.5, "description": "客户承诺还款"}
    ]
  }
}
```

## 数据处理输出

### 1. 转写结果
**格式**: JSON, 包含时间戳、说话人

```json
{
  "case_id": "case_001",
  "transcript": [
    {
      "start": 0.5,
      "end": 2.3,
      "speaker": "agent",
      "text": "Selamat siang, Bapak/Ibu..."
    },
    {
      "start": 2.5,
      "end": 5.1,
      "speaker": "customer",
      "text": "Ya, ada apa?"
    }
  ]
}
```

**位置**: `data/processed/transcripts/`

### 2. 结构化对话
**格式**: JSON, 对话级别特征

```json
{
  "case_id": "case_001",
  "metadata": {...},
  "turns": [
    {
      "turn_idx": 0,
      "speaker": "agent",
      "text": "...",
      "start_time": 0.5,
      "end_time": 2.3,
      "duration": 1.8,
      "tokens": [...],
      "intent": "greeting",
      "entities": {...},
      "sentiment": "neutral"
    }
  ],
  "dialogue_features": {
    "total_turns": 45,
    "agent_turns": 23,
    "customer_turns": 22,
    "total_duration": 245.3,
    "interruptions": 5,
    "agent_interruptions": 3,
    "customer_interruptions": 2
  }
}
```

**位置**: `data/processed/dialogues/`

---

## 数据预处理步骤

1. **格式转换** → 统一转为16kHz单声道WAV
2. **说话人分离** → 区分agent/customer
3. **语音转写** → 生成带时间戳的文本
4. **质量检查** → 人工审核低置信度转写
5. **结构化** → 提取特征、语义分析
