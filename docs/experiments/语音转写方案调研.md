# 本地语音转写（Voice to Text）方案调研

## 目标
在本地环境进行印尼语和英语的语音转写，支持：
- 长音频处理
- 说话人分离 (Speaker Diarization)
- 词级时间戳
- 多语言（印尼语 id，英语 en）

---

## 方案对比

### 1. OpenAI Whisper (开源)

| 特性 | 说明 |
|------|------|
| **模型** | tiny, base, small, medium, large, large-v2, large-v3 |
| **语言** | 支持 99 种语言，包括印尼语 |
| **输出** | 支持词级时间戳 |
| **速度** | large-v3 较慢，但 medium/small 可接受 |
| **安装** | `pip install openai-whisper` + ffmpeg |
| **GitHub** | https://github.com/openai/whisper |

**推荐**: Whisper large-v3 质量最好，但速度较慢；可以用 medium 做快速测试。

---

### 2. Faster-Whisper

| 特性 | 说明 |
|------|------|
| **基于** | Whisper，用 CTranslate2 重实现 |
| **速度** | 比原版快 2-4 倍 |
| **显存** | 占用更少 |
| **安装** | `pip install faster-whisper` |
| **GitHub** | https://github.com/guillaumekln/faster-whisper |

**推荐**: 优先使用，速度更快。

---

### 3. WhisperX

| 特性 | 说明 |
|------|------|
| **优势** | 支持词级对齐、说话人分离（集成 pyannote） |
| **GitHub** | https://github.com/m-bain/whisperX |

---

### 4. SpeechBrain

| 特性 | 说明 |
|------|------|
| **功能** | 完整的语音工具包，包括分离、识别 |
| **GitHub** | https://github.com/speechbrain/speechbrain |

---

### 5. 说话人分离（单独）

- **pyannote.audio**: https://github.com/pyannote/pyannote-audio
  - 需要 HuggingFace token
  - 质量最好
- **SpeechBrain**: 内置说话人分离
- **WhisperX**: 集成了 pyannote

---

## 推荐方案

### Phase 1: 仅转写（先试这个）
**方案**: Faster-Whisper (large-v3)
- 速度快
- 质量好
- 支持多语言

### Phase 2: + 说话人分离
**方案**: Faster-Whisper + pyannote.audio
- 先用 Whisper 转写
- 再用 pyannote 做说话人分离
- 后处理合并结果

### 简单方案: WhisperX
- 一键完成转写 + 对齐 + 分离
- 但安装可能更复杂
