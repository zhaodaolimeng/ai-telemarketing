# 分析方法

## 一、语音转写与说话人分离

### 转写方案
- **模型**: Whisper Large V3 / Faster-Whisper
- **语言**: Indonesian + English
- **输出**: 词级时间戳
- **置信度过滤**: 低置信度片段标记，人工审核

### 说话人分离
- **方法**: Pyannote.audio / SpeechBrain
- **目标**: 2个说话人 (agent, customer)
- **后处理**: 合并短片段、修正角色

---

## 二、会话状态发现方法

### 方法A: 无监督聚类（零标注启动）
1. **特征提取**
   - 句嵌入 (Sentence-BERT)
   - 对话历史上下文
   - 说话人角色
   - 时序位置（在对话中的位置）

2. **降维**
   - UMAP / t-SNE

3. **聚类**
   - HDBSCAN（自动发现类别数）
   - K-means（人工指定k）

4. **人工解读**
   - 查看每个簇的样本
   - 命名状态
   - 定义状态转移规则

### 方法B: 半监督（部分标注）
- 先标注一小部分数据
- 训练分类器
- 预测其余数据
- 人工检查不确定样本

### 方法C: 基于规则+LLM辅助标注
- LLM阅读对话，标注每个utterance的状态
- 人工审核和修正
- 迭代优化

## 三、话术有效性分析方法

### 核心思路
- **分组**: 成功回款 vs 未回款
- **对比**: 两组对话中话术的差异
- **统计**: 哪些话术在成功组显著更多

### 具体方法

#### 1. 话术相似度检索
```
对于每个话术U:
  计算与所有成功案例中话术的相似度
  计算与所有失败案例中话术的相似度
  计算差值 (成功相似度 - 失败相似度)
  → 正值: 倾向成功; 负值: 倾向失败
```

#### 2. n-gram 差异分析
- 提取成功组高频 n-gram
- 提取失败组高频 n-gram
- 计算差异显著性

#### 3. 话术模板提取
- 从成功案例中聚类相似话术
- 人工解读为模板
- 验证模板在独立测试集的效果

#### 4. 因果推断（更严谨）
- 匹配相似客户
- 对比用了话术A vs 话术B的效果
- 计算平均处理效应 (ATE)

### 统计检验
- Fisher精确检验
- 卡方检验
- Bootstrap置信区间

## 四、对话时序分析

### 状态转移图
- 从数据中学习状态转移矩阵
- 发现从"抵触"到"承诺"的关键路径

### 关键转折点识别
- 查找对话中情绪/意图突变的点
- 关联这些点前后的话术
- 分析什么触发了态度转变

### 打断分析
- 统计打断发生的位置、谁打断谁
- 对比成功/失败对话的打断模式
- 找出最佳打断时机模式

---

## 五、特征工程

### 话术级特征
- 文本嵌入
- 语义标签（施压、共情、协商...）
- 情感/情绪
- 具体度/模糊度
- 问句 vs 陈述句
- 礼貌程度

### 对话级特征
- 状态转移序列
- 各状态停留时间
- 谁主导对话
- 打断次数和位置
- 词汇丰富度
- 语速、停顿
