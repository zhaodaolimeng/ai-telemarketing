## ✅ 高成功率CTM的关键特征

### 1. 适当的停顿 (...)
- **CTM-040**: 19次使用
- **低成功率CTM**: 0次
- 👉 **关键差异**！适当的停顿让客户有思考时间

### 2. 多次确认 (Ya, ya, ya)
- **CTM-039**: 使用 "Ya, ya, ya."
- 👉 多次确认增强客户承诺

### 3. 简洁开场白
- "Halo?"
- "Halo."
- 👉 不啰嗦，直接开始

### 4. 礼貌结尾
- "Terima kasih."

---

## ❌ 低成功率CTM的问题

### 1. 只用 "Oke." 回应
- 过于简短，没有后续

### 2. 让客户说 "Sibuk." (忙)
- 说明没有成功说服客户

### 3. 开场白太长
```
Halo, Pak. Saya dari aplikasi Extra Uang Now.
Halo, Bapak. Bapak, jangan kurang-kurang mendengar, Pak.
```
- 👉 太长了，容易引起反感

### 4. 固定时间 ("Jam 4, bisa.")
- 建议让客户说时间，而不是直接指定

## 📋 智能语音话术建议（基于CTM-039和CTM-040的成功经验）

### 推荐流程
```
1. Halo?
2. Halo, selamat pagi/siang/sore Pak/Bu.
3. Saya dari aplikasi Extra.
4. Kapan bisa bayar?
5. [适当停顿...]
6. Oke, jam [X] ya Pak/Bu. Saya tunggu.
7. Ya, ya.
8. Terima kasih.
```

### 关键要点
- ✅ 使用 "Halo?" / "Halo."
- ✅ 适当停顿 (...)
- ✅ 多次确认 (ya, ya, ya)
- ✅ 让客户说时间
- ✅ 简洁开场白
- ✅ 礼貌结尾

### 避免
- ❌ 开场白太长
- ❌ 只用 "Oke." 回应
- ❌ 直接指定时间
- ❌ 过度强调应用名
