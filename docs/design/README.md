## 快速查看

### 核心技术选型
- **TTS**：Coqui TTS (VITS) - 高质量语音合成
- **ASR**：Faster-Whisper - 快速准确的语音识别
- **NLU**：分阶段 - 规则→轻量LLM
- **后端**：Python + FastAPI
- **语言**：印尼文、英文

### 架构路线
- **阶段1**：MVP，验证核心语音能力
- **阶段2**：优化，提升音质和准确率
- **阶段3**：扩展，多语种和能力增强
