## 快速浏览

### 核心模块
1. **任务管理** - 外呼任务创建与管理
2. **外呼引擎** - SIP对接、自动外呼
3. **对话引擎** - TTS/ASR/NLU、对话管理
4. **话术管理** - 话术库、模板、A/B测试
5. **录音质检** - 录音、转写、智能质检
6. **数据管理** - 客户数据、通话记录、报表
7. **系统管理** - 用户、权限、配置

### 技术关键点
- 语音合成与识别
- 自然语言理解
- 对话流程编排
- 并发控制
- 数据安全与合规

### 催收核心能力
- 超低延迟响应（≤300ms）
- 智能打断与状态检测
- 用户画像与标签体系
- 动态话术选择与调整
- 施压策略分层（M0/M1/M2+）
- 催收效果评估与优化闭环
