## 快速浏览

### 核心模块
1. **任务管理** - 外呼任务创建与管理
2. **外呼引擎** - SIP对接、自动外呼（规划中）
3. **对话引擎** - TTS/ASR/NLU、对话状态机、LLM Fallback混合架构
4. **话术管理** - 话术库、模板、变量替换
5. **录音质检** - 录音、转写、智能质检（规划中）
6. **数据管理** - 客户数据、通话记录、报表分析
7. **系统管理** - 用户、权限、配置（规划中）

### 技术关键点
- 印尼语语音合成与识别
- 自然语言理解与语义解析
- 状态机驱动的对话流程管理
- LLM兜底混合架构
- 超低延迟响应（≤300ms）
- 智能打断与实时交互
- 数据安全与合规

### 催收核心能力
- 分层施压策略（H2宽限期前2天温和引导 / H1宽限期前1天引导+暗示后果 / S0实质性逾期高压催收）
- 用户画像与标签体系
- 动态话术选择与调整
- 智能打断与状态实时检测
- 催收效果多维度评估与优化闭环
