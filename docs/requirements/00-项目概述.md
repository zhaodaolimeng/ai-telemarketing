# 项目概述

## 项目名称
智能语音催收系统（AI-Telemarketing）

## 项目简介
智能语音催收系统是一个面向印尼信贷催收场景的自动化语音外呼与智能对话系统。通过整合语音合成、语音识别、自然语言理解等AI技术，实现自动化的债务提醒、还款协商、客户服务等功能。

系统采用"规则引擎为主，LLM兜底为辅"的混合架构，在保证高性能和低成本的同时，具备处理复杂场景的能力。

## 应用场景
### 主要场景
1. **贷后提醒** - 还款日之前提醒客户按时还款
2. **逾期催收** - 联系逾期客户了解情况，协商还款计划
3. **信息确认** - 确认客户联系方式变更、身份信息等
4. **满意度调查** - 业务完成后回访客户，收集反馈
5. **产品营销** - 向客户介绍新产品和服务

### 覆盖催收阶段

| 业务阶段 | 代码 | 说明 | 催收策略 |
|---------|------|------|----------|
| **宽限期前2天** | H2 | 宽限期到期前2天，非实质性逾期 | 温和引导，提醒+协助客户正常结清或展期 |
| **宽限期前1天** | H1 | 宽限期到期前1天，非实质性逾期 | 引导+暗示后果，加强紧迫感但不高压施压 |
| **实质性逾期** | S0 | 已过宽限期，实质性逾期 | 高压催收，明确告知后果，坚持要求具体还款时间 |

## 项目目标

### 阶段1: MVP验证（已完成）
1. 实现基础的对话功能
2. 支持单轮和多轮对话
3. 集成TTS语音合成
4. 实现对话记录和存储
5. 14个基础场景成功率达到85%以上

### 阶段2: 能力增强（进行中）
1. 支持智能打断和实时语音交互
2. 实现LLM Fallback混合架构，处理复杂场景
3. 接入真实ASR语音识别服务
4. 实现红黑对抗训练框架，持续提升鲁棒性
5. 支持小范围生产测试

### 阶段3: 生产落地（规划中）
1. 接入真实SIP电话线路，支持大规模外呼
2. 全渠道接入（电话、短信、微信等）
3. 智能质检和合规性检查
4. 预测性外呼策略优化
5. 机器人训练和优化闭环
6. 支持多语种和多场景扩展

## 核心价值
1. **降低人力成本** - 机器人替代人工进行重复性工作
2. **提升效率** - 支持大规模并发外呼
3. **标准化执行** - 话术和流程标准化，减少人为偏差
4. **数据完整** - 完整录音和对话记录，便于质检
5. **全天候服务** - 7x24小时不间断服务
